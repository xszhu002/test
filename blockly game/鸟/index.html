积木X 代表X 轴上鸟的飞行方向，鸟巢所在位置为X 轴上的80 单位，当鸟的飞行位置还没到达80 单位时( 也就是X 轴小于80 ) ，让他持平往右边飞( 0 ° ) ，当飞行到鸟巢所在的80 单位后( 也就是X 轴大于80 时) ，改变航道往下飞( 270° ) 。
